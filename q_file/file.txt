- 파일 -
    외부에 파일을 생성하거나 내용을 작성할 수 있으며,
    기존의 내용도 읽어올 수 있다.

- 파일 생성 -
    파일을 열고 작성할 때 사용
    'w'를 작성해서 운정체제에게 파일을 여는 목적(write)을 알려줘야 된다

    open(path, 'w')     -> 열린 파일 객체


- 내용 추가 -
    기존의 내용을 없애지 않고, 그 이래에 내용을 추가한다
    이 때, 추가(add) 모드라는 뜻의 'a'를 사용한다

    open(path, 'a')


- 파일 읽기 -
    기존 네용을 한 줄씩 읽어올 때 'r'을 사용해서 읽기(read) 모드로 파일을 열어준다
    open(path, 'r')
